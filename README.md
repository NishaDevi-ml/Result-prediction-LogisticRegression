# Student Performance Prediction Using Logistic Regression

## Project Overview

This project uses Logistic Regression to predict whether a student is likely to pass or fail based on study hours and academic performance indicators.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn
* Jupyter Notebook

## Workflow

1. Data Loading
2. Data Exploration and Visualization
3. Data Preprocessing
4. Train-Test Split
5. Logistic Regression Model Training
6. Prediction
7. Model Evaluation

## Machine Learning Algorithm

Logistic Regression

## Key Features

* Exploratory Data Analysis (EDA)
* Data Visualization
* Binary Classification
* Model Training and Prediction
* Performance Evaluation

## Results

The model classifies students into pass or fail categories based on study-related factors and helps understand the relationship between study habits and academic outcomes.
